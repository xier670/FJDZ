# 开发工具准备
操作系统：win10

开发工具：pycharm

python版本：python3.6(virtualenv虚拟独立环境)

python内置模块：sys,random,codecs

第三方模块：pygame1.9.6

# 功能介绍
1)	敌机：绘制图片，移动敌机，敌机爆炸动画。

2)	玩家飞机：发射子弹，绘制图片，玩家爆炸动画。

3)	排行榜：保存分数到文件，更新文档分数排行，读取分数文件，显示文字。
